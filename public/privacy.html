Pridebot Privacy Policy

Last Updated: August 22, 2024

1. Data Collection

Pridebot collects the following data:

  • User IDs:
    • Server Logging: To track and log user actions within servers for moderation and troubleshooting.
    • Command Logging: To monitor the usage of commands and analyze how the bot is used to improve functionality.
    • Profile Filing: To store and manage user profiles, including custom fields.
    • Pride Avatar Filing: To store and manage users avatars.
    • Bot Votes: To record and display the number of votes received on bot listing platforms.
  • Server IDs:
    • Server Logging: To track and log activities across different servers where Pridebot is active, helping with analytics and moderation.
    • Command Logging: To keep track of command usage within each server to tailor features and support.
  • User Profile Pictures:
    • Pride Avatar Creation: To allow users to create customized pride avatars based on their profile pictures.
  • User Profile Information:
    • Profile Display: To allow users to customize and display their profiles, including pronouns, sexual orientation, romantic orientation, and gender identity.

2. Use of Data

The collected data is used for the following purposes:

  • Security:
    • Monitoring and ensuring the security of the services by tracking user and server activity to prevent abuse, detect suspicious behavior, and maintain a safe environment.
  • Development:
    • Improving the features and functionality of Pridebot based on user interaction data, including command usage statistics and user feedback, to provide a better user experience.
    • Enhancing profile and avatar features by analyzing how users engage with these tools, allowing for targeted updates and new functionalities.
  • Client Usage:
    • Enabling users to interact with Pridebot effectively, ensuring that commands, features, and services function as intended and meet user needs.
    • Understanding how commands and websites associated with Pridebot are used, enabling the optimization of services and ensuring that they meet user needs.

3. Data Sharing

We do not share your data with third parties except in the following circumstances:

  • When required by law or to respond to legal processes, such as subpoenas or court orders.
  • To protect the rights, property, or safety of our users, Pridebot, or others, including enforcing our Terms of Service and preventing fraud or security breaches.

4. Data Security

All User IDs, Server IDs and User Profile Information is securely stored in MongoDB encrypted database. All Profile Pictures are stored in bot files which can only be accessed by Sdriver1. However, please note that no security system is impenetrable, and we cannot guarantee the absolute security of your data. We continuously review and enhance our security practices to safeguard your information.

5. Data Deletion

If you wish to have your data deleted, please contact Sdriver1 or in the Support Server. We will process your request and delete your data from our records.

6. Changes to the Privacy Policy

We reserve the right to update this Privacy Policy at any time. Your continued use of Pridebot after any changes will constitute your acceptance of the new terms.

7. Contact Us

If you have any questions about this Privacy Policy, please contact Sdriver1 or ask for help in the Support Server.
